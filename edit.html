How to edit the pages in https://icyphy.github.io/ptII/

The static pages like this one are edited via the gh-pages branch of the Github ptII repo:

      git clone --depth=50 --branch=master --single-branch --branch=gh-pages https://github.com/icyphy/ptII gh-pages

See https://pages.github.com/ for details.

The pages in the logs and downloads directory are created as part of the Travis-ci ptII build, which runs $PTII/bin/ptIITravisBuild.sh.
