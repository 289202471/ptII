Windows 32-bit vs 64-bit

The difference between the Windows 32-bit and 64-bit installers in these devel versions has to do with how shortcuts are created, see IZPack throws exception while creating Windows 7 shortcuts.

Under Windows, to determine whether you are running a 32-bit version or a 64-bit version of windows:

  • Windows 7 or Vista: "Start", then right click on "Computer", then click on "Properties". Under System, you should see the system type. The 64-bit version of Windows 7 will say "64-bit Operating System". See http://windows.microsoft.com/en-US/windows7/32-bit-and-64-bit-Windows-frequently-asked-questions for details.
  • Windows XP: "Start", then right click on "My Computer", and then click on "Properties". Under "System", if you see "x64 Edition", then you are running a 64-bit version of Windows. If "x64 Edition" is not listed, then you are running a 32-bit version of Windows.

How to decide which snapshot to download

The Ptolemy II tree is under active development. Rather than downloading a snapshot of our tree which may contain bugs that require fixing, we encourage you to use Git so that you will always have the latest version. See the ptexternal page for SVN instructions.

Before downloading a nightly build installer, please check the last build status and see that it completed. See the Jenkins ptII Nightly Build page for the status of recent builds.
